Nexly Support

Last Updated: July 13, 2026

1. About Nexly

Nexly is a Discord server management and automation platform developed and maintained by Nexly Labs. Nexly provides tools designed to help communities manage, customize, and automate their Discord servers.

2. Getting Help

If you need help with Nexly, experience an issue, or have questions about the service, you can contact the Nexly Labs team using the available support channels.

3. Bug Reports

If you discover a bug or unexpected behavior in Nexly, please provide as much information as possible, including:

  • Description of the problem
  • Steps required to reproduce the issue
  • Relevant error messages
  • Discord server information if necessary

Detailed reports help us investigate and resolve issues faster.

4. Security Reports

If you discover a potential security vulnerability in Nexly, please contact Nexly Labs privately. Do not publicly share security issues before they have been reviewed.
